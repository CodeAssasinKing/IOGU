ÝAGŞYGELDI KAKAÝEW ADYNDAKY HALKARA NEBIT WE GAZ UNIWERSITETI

Dalaşgär — 2024: geljegiň binýady şu günden tutulýar

2024/2025-nji okuw ýyly üçin aşakdaky hünärler (hünärmen maksatnamasy) we taýýarlyk ugurlary (bakalawr maksatnamasy) boýunça talyplyga kabul edýär:

I. Hünärmen maksatnamasy:

  1. Amaly geologiýa, dag işi, nebitgaz işi we geodeziýa
    • nebit-gaz känlerini özleşdirmek we ulanmak
    • nebit we gaz guýularyny burawlamak
    • nebitgazgeçirijilerini we nebitgazyň saklanýan ýerlerini taslamak, gurmak we ulanmak
    • nebitiň we gazyň geologiýasy
    • gidrogeologiýa we inžener geologiýasy
    • geologiki kartalaşdyrma we magdan ýataklarynyň gözlegi
    • peýdaly magdanlary gözlemegiň we barlamagyň geofiziki usullary
    • markşeýder işi
    • gazylyp alynýan peýdaly magdanlary baýlaşdyrmak
  2. Himiýa tehnologiýalary
    • nebiti we gazy gaýtadan işlemegiň himiki tehnologiýasy
  3. Tehnosfera howpsuzlygy we tebigaty abadanlaşdyryş
    • daşky gurşawy goramak we tebigy baýlyklary tygşytly peýdalanmak
  4. Maşyngurluşyk
    • sowadyjy, kompressor maşynlary we desgalary
  5. Gurluşygyň tehnikasy we tehnologiýalary
    • nebitgaz desgalary we senagat gurluşygy
  6. Elektrik we ýylylyk energetikasy
    • senagat desgalarynyň we tehnologik toplumlaryň elektrohereketlendirilişi hem-de awtomatikasy (nebitgaz pudagynda)
    • gazturbina desgalary we gazkompressor stansiýalary

II. Bakalawr maksatnamasy:

  1. Amaly geologiýa, dag işi, nebitgaz işi we geodeziýa (dag işi)
  2. Tehniki ulgamlarda dolandyryş (standartlaşdyrmak we metrologiýa)
  3. Informatika we hasaplaýyş tehnikasy
    • maglumat ulgamlary we tehnologiýalary
    • kompýuter ulgamynyň programma üpjünçiligi
  4. Maglumat howpsuzlygy
  5. Maşyngurluşyk (tehnologiki maşynlar we enjamlar)
  6. Elektrik we ýylylyk energetikasy (elektroenergetika we elektrotehnika)
  7. Ykdysadyýet we dolandyryş
    • nebitgaz pudagynyň ykdysadyýeti we dolandyrylyşy
    • işgärleri dolandyrmak

III. Bakalawr maksatnamasy (iňlis dilinde):

  1. Amaly geologiýa, dag işi, nebitgaz işi we geodeziýa
    • nebitgaz işi
    • magdanlaryň geologiýasy we gözlegi
  2. Himiýa tehnologiýalary (himiýa tehnologiýasy)
  3. Informatika we hasaplaýyş tehnikasy (programma inženerligi)
  4. Ykdysadyýet we dolandyryş (logistika)
